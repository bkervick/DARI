# DARI
Rim protection model

How to run:
Get csv file with list of players with team info, block rate, and foul rate (in form fouls per 50) and name it players.csv. You can get this information by downloading the data from Hoop-Explorer via csv. Or use players.csv from this repository.

Run get_rim_protection_metrics.py.

Then edit dari_model_v2.py to include your KenPom login information. Then run dari_model_v2.py.
